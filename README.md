# LinkyEngine
How do I build a 3D game engine?

## Build & Install

```bash
$ cmake -S ./ -B ./build
$ cmake --build ./build --config release
```
## Project Architecture

## Dependencies

- C++ 20
- CMake
- C++ compiler (MSVC, Mingw, ...)
- Glad (2.0.2)
- GLFW (3.3.8)
- OpenGl (3.3)
- Dear ImGui (1.88)
- glm (0.9.8.5)
- stb_image
- Python (3.11.1) & jinja2
